# Bike Info

#### This application searches a database of bikes based on location and date stolen

#### By Suzanne Schuber & Natalie Benjes

## Technologies Used

* _HTML_
* _JS_
* _CSS_


## Description
* _This application takes in user inputs for date stolen and location, then accesses the bike index API._

## Setup/Installation Requirements


* Installing all packages with $ npm install.
* Building the project using webpack with $ npm run build
* Starting a development server with $ npm run start
* Linting JS files in the src folder with $ npm run lint
* Running tests with Jest using $ npm run test
